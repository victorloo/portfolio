About

剛踏入 Web 世界的 junior Web Developer

在學期間有顯著的網路成癮行為，且熱愛挖掘網路能提供什麼體驗。服役後，經歷過一段工作與迷惘的經歷，之後便下定決心朝網路業發展，一個充滿著挑戰以及成就感的工作環境。

在線上學習課程︰ Alpha Camp 大航道計畫中，除了基本的技術能力(前端、後端、GIT、API、佈署等)外，也教導產品思維，了解工程師除了基本功要打穩，更要貼身了解產品是因為有需求才需要被開發，讓我的思緒慢慢跳脫以往的視野。

上課期間培養自學的習慣，尋找各種文章來增進自己的視野，例如︰

  • 學習產品思維後閱讀 UX 的書好初步了解使用者想要的體驗。
  • 學習基礎 JavaScript 後，前往 Udemy 學習︰JavaScript 全攻略：克服 JS 的奇怪部分。了解更多JS的設計原理。
  • 後端課程告一段落後，基於對前端框架好奇，按照 React tutorial 打造出第一個 React 作品，在過程中觀察 MPA 和 SPA 的不同。

課程最後組隊撰寫自主專案，過程中發覺自己喜歡前端，也適應前端的開發過程(Chrome devTool真是棒透了)。於是在專案結束後，下定決心朝前端工程師的方向邁進。

一連串的學習下、實作後，了解網路產業的變動快，新技術可能明年就被淘汰。因為有著嚴峻的變化，反而令人覺得有挑戰。完成基本課程的我，對投身這塊領域的自己充滿期許！

目前製作 React 作品中，找尋前端工程師的工作機會
